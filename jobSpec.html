JOB SPECIFICATION
If you want to work as a web developer at Hot Beans Web, you would need the following qualifications to show that you have some background knowledge.

This includes:
- GCSE in IT (or BTEC equivalent)
- GCSE in Art (or BTEC equivalent)
- Level 2 award in Creating a Website
- Level 2 Certificate in Interactive Media
- Level 2/3 Diploma in Creative iMedia
- Level 3 Diploma in Information Technology (Specialist)
- Level 3 Diploma in IT User Skills


You will work normal office hours, 9am to 5pm, Monday-Friday. We are also flexible and can allocate more work hours if it is needed. Work can be completed from home if you prefer it that way.
